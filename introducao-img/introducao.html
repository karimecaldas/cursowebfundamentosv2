Trabalhando com imgaens

Imagens vetoriais vs bitmap

Imagens vetoriais são criadas a partir de um modelo geométrico, enquanto imagens bitmap são armazenadas como matrizes de pixels com valores de cor.


Imagens transparentes

Alguns navegadores não suportam imagens transparentes. Para torná-las visíveis, você pode usar uma imagem PNG com fundo transparente.


Gif animado

Gifs animados são utilizados para mostrar efeitos animados, como movimento, mudança de cor ou efeito de explosão.

Float

Float é um método de posicionamento que permite que um elemento seja posicionado à esquerda ou à direita de outro elemento na página.
